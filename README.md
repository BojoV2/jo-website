# PDF Word Insert & Workflow Management System

Docker-ready backend for:
- Role-based auth (`super_admin`, `admin`, `user`)
- PDF template upload + field coordinate configuration
- PDF generation by coordinate insertion
- Per-template workflow tabs (`pending`, `done`, `cancelled`, `rescheduled`)
- Frontend web app (login, admin console, user portal)

See `docker-compose.yml` and `backend/README.md`.

System flow diagrams: `docs/SYSTEM_FLOWS.md`
